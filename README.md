# ❌⭕ Tic Tac Toe Game

A simple and interactive **Tic Tac Toe game** built using **HTML, CSS, and JavaScript**.  
This project demonstrates core JavaScript logic, DOM manipulation, and game state handling.

---

## 📌 Live Demo

🌐 https://tic-tac-toe-game-seven-silk.vercel.app

---

## 💻 Technologies Used

- HTML5  
- CSS3  
- JavaScript (ES6)

---

## 🎯 Features

- 🎮 Two-player game (X vs O)  
- 🧠 Win detection logic  
- 🔄 Game reset functionality  
- 🎨 Simple and clean UI  
- 📱 Responsive design (basic)  

---

## 📁 Project Structure

- `index.html` → Game UI  
- `style.css` → Styling  
- `app.js` → Game logic  

---

## 🚀 How to Play

1. Player X starts the game  
2. Players take turns clicking boxes  
3. First to get 3 in a row wins  
4. Click reset to restart game
